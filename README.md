# Counter App

A simple counter application built using HTML, CSS, and JavaScript.

## Features

- Increase counter
- Decrease counter
- Reset counter
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This project was created to practice:
- DOM manipulation
- Git and GitHub workflow
- Branching and merging

## How to Run

1. Clone the repository
2. Open "index.html" in your browser
